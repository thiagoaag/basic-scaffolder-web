# basic-scaffolder-web

An simple folder structure for web projects
```sh
project-name
    |---- src 
            |---- fonts
            |---- html
                    |---- components
                    |---- includes
                    |---- pages
            |---- images
            |---- js
                    |---- components
                    |---- pages
                    |---- services
                    |---- templates
                    |---- main.js
            |---- css
            |---- third-party
    files_config.json
    files_config.js
```
